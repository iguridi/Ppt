Readme

- Este programa se encarga de hacer un ppt automáticamente
- Para correrlo se debe ejecutar datos.py, de la forma "python datos.py [titulo] [web]". 
Si se pone algo en web se considera que toma los datos automáticamente de la página 
eucaristíadiaria.cl

Para correrlo en windows sin error, se debe instalar un programa y luego correrlo tal como se indica a continuación

T:\> py -mpip install win-unicode-console
T:\> py -mrun your_script.py
